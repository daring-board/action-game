# 仕様

*   **ゲームの目的:**
    *   プレイヤーがキャラクターを操作し、画面上を動く敵をよけるシンプルなゲームです。
*   **使用技術:**
    *   **フレームワーク:** Next.js (React/TypeScript)
    *   **描画:** HTMLのDOM要素とCSS
    *   特別なゲームライリは使わず、Reactの機能で基本的なゲームループとインタラクションを実装します。
*   **主な機能:**
    *   **プレイヤー:**
        *   四角形で表現します。
        *   キーボードの矢印キー（左/右）で操作できます。
    *   **敵:**
        *   別の色の四角形で表現します。
        *   画面の上から下へ自動で落下してきます。
    *   **当たり判定:**
        *   プレイヤーと敵が接触したらゲームオーバーとします。
